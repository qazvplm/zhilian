# 智联投递助手

智联招聘（zhaopin.com）自动投递 Chrome 扩展。在搜索结果页自动点击"立即投递"按钮，支持投递总数、间隔时间、排除词等配置。

## 功能

- **自动投递**：在智联搜索结果页自动点击"立即投递"
- **投递总数**：设置最大投递数量，到达后自动停止
- **投递间隔**：可配置每次投递之间的等待秒数，避免触发风控
- **排除词过滤**：支持逗号分隔的排除关键词（如：外包、培训、销售），匹配到的职位自动跳过
- **跳过已投递**：自动识别已投递/已申请的职位并跳过
- **自动翻页**：当前页投递完毕后自动进入下一页
- **安全验证检测**：检测到验证码或登录弹窗时自动暂停
- **可拖拽面板**：浮窗支持拖拽移动和收起

## 安装

1. 下载本仓库代码
2. 打开 Chrome，访问 `chrome://extensions/`
3. 开启右上角「开发者模式」
4. 点击「加载已解压的扩展程序」，选择本仓库文件夹
5. 扩展图标会出现在工具栏

## 使用方法

1. 打开 [智联招聘](https://www.zhaopin.com/) 并登录
2. 搜索职位并设置筛选条件（城市、薪资、学历等）
3. 在搜索结果页面，右侧会出现「智联投递助手」浮窗
4. 配置投递参数：
   - **投递总数**：本次最多投递多少个职位（默认 30）
   - **投递间隔**：每次投递后等待的秒数（默认 8 秒）
   - **排除词**：包含这些关键词的职位会被跳过
5. 点击「开始投递」
6. 随时可以点击「停止」中断

## 注意事项

- 请先在智联网站上自行搜索和筛选，插件只按当前搜索结果进行投递
- 建议设置合理的投递间隔（5-10 秒），过快可能触发平台风控
- 遇到验证码或登录提示时，插件会自动暂停，请手动处理后重新开始
- 投递完成后，成功页面会自动关闭
后果由用户自行承担，请遵守智联招聘平台的使用条款。
